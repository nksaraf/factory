# Navigation

Components for navigating between views, organizing content into tabs, and presenting contextual actions.

```tsx
import { Tabs, TabsContent, TabsList, TabsTrigger } from "@rio.js/ui-next/components/tabs"
```

---

## Tabs

Switch between different views or sections within the same page context.

<ComponentPreview className="block">

      Overview content goes here.

      Analytics content goes here.
    </TabsContent>
    <TabsContent value="settings" className="mt-4 text-sm">
      Settings content goes here.

<Tabs defaultValue="overview">

Tabs support three visual variants: `default` (filled background), `outline` (underline), and `pills` (rounded pill active state).

### Controlled tabs

```tsx
const [tab, setTab] = useState("overview")

<Tabs value={tab} onValueChange={setTab}>
  <TabsList>
    <TabsTrigger value="overview">Overview</TabsTrigger>
    <TabsTrigger value="analytics">Analytics</TabsTrigger>
  </TabsList>
  <TabsContent value="overview">...</TabsContent>
  <TabsContent value="analytics">...</TabsContent>
</Tabs>
```

### Tab structure

| Component | Purpose |
|-----------|---------|
| `Tabs` | Root — manages active tab state |
| `TabsList` | Container for tab triggers (the tab bar) |
| `TabsTrigger` | Clickable tab label |
| `TabsContent` | Panel content shown when its tab is active |

## Breadcrumb

A trail showing the user's location in a hierarchy. Use for multi-level navigation where users need to backtrack.

```tsx
import { Breadcrumb, BreadcrumbItem, BreadcrumbLink, BreadcrumbList, BreadcrumbPage, BreadcrumbSeparator } from "@rio.js/ui-next/components/breadcrumb"
```

<ComponentPreview>
  <Breadcrumb>
    <BreadcrumbList>
      <BreadcrumbItem>
        <BreadcrumbLink href="/">Home</BreadcrumbLink>
      </BreadcrumbItem>
      <BreadcrumbSeparator />
      <BreadcrumbItem>
        <BreadcrumbLink href="/docs">Docs</BreadcrumbLink>
      </BreadcrumbItem>
      <BreadcrumbSeparator />
      <BreadcrumbItem>
        <BreadcrumbPage>Navigation</BreadcrumbPage>
      </BreadcrumbItem>
    </BreadcrumbList>
  </Breadcrumb>
</ComponentPreview>

```tsx
<Breadcrumb>
  <BreadcrumbList>
    <BreadcrumbItem>
      <BreadcrumbLink href="/">Home</BreadcrumbLink>
    </BreadcrumbItem>
    <BreadcrumbSeparator />
    <BreadcrumbItem>
      <BreadcrumbLink href="/docs">Docs</BreadcrumbLink>
    </BreadcrumbItem>
    <BreadcrumbSeparator />
    <BreadcrumbItem>
      <BreadcrumbPage>Current Page</BreadcrumbPage>
    </BreadcrumbItem>
  </BreadcrumbList>
</Breadcrumb>
```

### Breadcrumb structure

| Component | Purpose |
|-----------|---------|
| `Breadcrumb` | Root `<nav>` with `aria-label="breadcrumb"` |
| `BreadcrumbList` | Ordered list container |
| `BreadcrumbItem` | Individual breadcrumb segment |
| `BreadcrumbLink` | Clickable link to a parent page |
| `BreadcrumbPage` | Current page (non-clickable, `aria-current="page"`) |
| `BreadcrumbSeparator` | Visual separator between items (default: `/`) |

### With React Router

```tsx
import { Link } from "react-router"

<BreadcrumbLink asChild>
  <Link to="/workspaces">Workspaces</Link>
</BreadcrumbLink>
```

## Dropdown Menu

A menu of actions or options triggered by clicking a button. Use for overflow menus, context actions, and navigation shortcuts.

```tsx
import { DropdownMenu, DropdownMenuContent, DropdownMenuItem, DropdownMenuSeparator, DropdownMenuTrigger } from "@rio.js/ui-next/components/dropdown-menu"
```

    <DropdownMenuContent align="end">
      <DropdownMenuItem>
        <span className="icon-[ph--user-duotone] mr-2 h-4 w-4" /> Profile
      </DropdownMenuItem>
      <DropdownMenuItem>
        <span className="icon-[ph--gear-duotone] mr-2 h-4 w-4" /> Settings

        <span className="icon-[ph--sign-out-duotone] mr-2 h-4 w-4" /> Sign out

    </DropdownMenuContent>
  </DropdownMenu>
</ComponentPreview>

### Sub-menus

```tsx
import { DropdownMenuSub, DropdownMenuSubContent, DropdownMenuSubTrigger } from "@rio.js/ui-next/components/dropdown-menu"

<DropdownMenuSub>
  <DropdownMenuSubTrigger>More options</DropdownMenuSubTrigger>
  <DropdownMenuSubContent>
    <DropdownMenuItem>Option A</DropdownMenuItem>
    <DropdownMenuItem>Option B</DropdownMenuItem>
  </DropdownMenuSubContent>
</DropdownMenuSub>
```

### Dropdown structure

| Component | Purpose |
|-----------|---------|
| `DropdownMenu` | Root — manages open/close state |
| `DropdownMenuTrigger` | Element that opens the menu |
| `DropdownMenuContent` | The floating menu panel |
| `DropdownMenuItem` | A single menu action |
| `DropdownMenuSeparator` | Visual divider between groups |
| `DropdownMenuLabel` | Non-interactive group heading |
| `DropdownMenuShortcut` | Right-aligned keyboard shortcut hint |
| `DropdownMenuSub` | Sub-menu container |

## API Reference

### Tabs Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `string` | — | Controlled active tab |
| `defaultValue` | `string` | — | Uncontrolled default tab |
| `onValueChange` | `(value: string) => void` | — | Called when active tab changes |
| `orientation` | `"horizontal" \| "vertical"` | `"horizontal"` | Layout direction |

### TabsList Variants

| Variant | Style |
|---------|-------|
| `default` | Rounded container with muted background |
| `outline` | Bottom border, no background |
| `pills` | No background or border, pill-shaped active state |

### TabsTrigger Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `string` | — | Unique identifier for this tab (required) |
| `disabled` | `boolean` | `false` | Disables the tab trigger |

### BreadcrumbLink Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `href` | `string` | — | Link destination |
| `asChild` | `boolean` | `false` | Render as child element (e.g., React Router `Link`) |

### DropdownMenuContent Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `align` | `"start" \| "center" \| "end"` | `"center"` | Horizontal alignment relative to trigger |
| `side` | `"top" \| "right" \| "bottom" \| "left"` | `"bottom"` | Preferred side |
| `sideOffset` | `number` | `4` | Distance from trigger in pixels |

### DropdownMenuItem Props

Supports RichProps for icons, tooltips, and permission gating.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `disabled` | `boolean` | `false` | Disables the menu item |
| `onSelect` | `(event: Event) => void` | — | Called when item is selected |
| `icon` | `string` | — | Iconify class for left icon |
| `iconRight` | `string` | — | Iconify class for right icon |
| `shortcut` | `string` | — | Keyboard shortcut hint (right-aligned) |
| `loading` | `boolean` | `false` | Shows spinner and disables item |
| `allowed` | `boolean` | — | Permission gate. `false` disables and shows `reason` |
| `reason` | `string` | — | Tooltip text when `allowed` is `false` |
| `inset` | `boolean` | `false` | Adds left padding to align with icon items |
