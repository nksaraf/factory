# Buttons

The `Button` component is the primary interactive element for triggering actions. It supports multiple visual variants, sizes, and rich props like icons, loading state, tooltips, and keyboard shortcuts.

```tsx
import { Button } from "@rio.js/ui/components/button"
```

## Variants

Use the `variant` prop to change the visual style of the button.

<VariantGrid>
  <Button variant="primary">Primary</Button>
  <Button variant="secondary">Secondary</Button>
  <Button variant="destructive">Destructive</Button>
  <Button variant="success">Success</Button>
  <Button variant="outline">Outline</Button>
  <Button variant="ghost">Ghost</Button>
  <Button variant="link">Link</Button>
</VariantGrid>

```tsx
<Button variant="primary">Primary</Button>
<Button variant="secondary">Secondary</Button>
<Button variant="destructive">Destructive</Button>
<Button variant="success">Success</Button>
<Button variant="outline">Outline</Button>
<Button variant="ghost">Ghost</Button>
<Button variant="link">Link</Button>
```

### When to use each variant

- **primary** — The main call-to-action on a page. Use sparingly (one per section).
- **secondary** — Secondary actions that complement the primary action.
- **destructive** — Dangerous actions like deleting resources. Pair with a confirmation dialog.
- **success** — Positive confirmations like "Approve" or "Complete".
- **outline** — Neutral actions with lower visual weight. Good for toolbars.
- **ghost** — Minimal buttons for inline actions, icon buttons, or dense UIs.
- **link** — Inline text actions that look like links but behave as buttons.

| Size | Height | Use case |
|------|--------|----------|
| `sm` | 32px | Dense UIs, table rows, toolbars |
| `md` | 36px | Default for most actions |
| `lg` | 40px | Hero sections, prominent CTAs |
| `icon` | 36x36px | Icon-only buttons (toolbar, controls) |

## Tooltips and Shortcuts

Use `tooltip` for hover hints and `shortcut` to display a keyboard shortcut badge.

</VariantGrid>

<Button tooltip="Create a new workspace" icon="icon-[ph--plus-bold]">

## Permission Gating

Use `allowed` and `reason` to gate actions behind permissions. When `allowed={false}`, the button is disabled and the `reason` is shown as a tooltip.

<VariantGrid>
  <Button allowed={false} reason="You need admin access to delete workspaces" variant="destructive" icon="icon-[ph--trash-duotone]">Delete Workspace</Button>
  <Button allowed={true} icon="icon-[ph--pencil-duotone]">Edit</Button>
</VariantGrid>

```tsx
<Button
  allowed={false}
  reason="You need admin access to delete workspaces"
  variant="destructive"
  icon="icon-[ph--trash-duotone]"
>
  Delete Workspace
</Button>
```

## API Reference

### Button Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `"primary" \| "secondary" \| "destructive" \| "success" \| "outline" \| "ghost" \| "link"` | `"default"` | Visual style |
| `size` | `"sm" \| "md" \| "lg" \| "icon"` | `"md"` | Button size |
| `icon` | `string` | — | Iconify icon class rendered before children |
| `iconRight` | `string` | — | Iconify icon class rendered after children |
| `loading` | `boolean` | `false` | Shows spinner and disables button |
| `tooltip` | `string \| ReactNode` | — | Tooltip content on hover |
| `shortcut` | `string` | — | Keyboard shortcut badge (e.g. `"⌘K"`) |
| `allowed` | `boolean` | — | Permission gate. `false` disables and shows `reason` |
| `reason` | `string` | — | Tooltip text when `allowed` is `false` |
| `description` | `string` | — | Fallback tooltip when `tooltip` is not set |
| `asChild` | `boolean` | `false` | Render as child element via Radix Slot |
| `disabled` | `boolean` | `false` | Standard HTML disabled state |

### ButtonGroup Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `orientation` | `"horizontal" \| "vertical"` | `"horizontal"` | Layout direction |

### ButtonGroupText Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `asChild` | `boolean` | `false` | Render as child element via Radix Slot |
